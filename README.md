# WallPrompt 📱🎨

WallPrompt is a modern Android wallpaper application built with Kotlin that allows users to browse, generate, and download high-quality wallpapers. The app combines curated collections with AI-powered wallpaper generation to provide users with unique and personalized wallpapers.

> Developed by Abdul Hanan

---

## ✨ Features

### 🖼️ Curated Wallpapers
- Browse professionally selected wallpapers
- Organized categories
- High-quality image previews
- Smooth fullscreen viewing experience

### 🤖 AI Wallpaper Generator
- Generate wallpapers from text prompts
- Multiple style options
- Aspect ratio selection
- Resolution selection
- Prompt enhancement before generation

### 📥 Downloads
- Save wallpapers directly to device
- Rewarded unlock system
- High-resolution downloads

### 👤 User Accounts
- Email authentication
- Personal generated wallpaper history
- Saved prompts
- User profile management

### 📱 Modern UI
- Material Design interface
- Responsive layouts
- Fast image loading
- Smooth animations
- Mobile-first experience

### 💰 Monetization
- Google AdMob integration
- Rewarded Ads
- Banner Ads
- Premium-ready architecture

# Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/wallprompt.git
```

2. Open in Android Studio.

3. Add your Firebase configuration:

```
google-services.json
```

4. Configure:

- Firebase Authentication
- Firestore
- Storage
- Cloud Functions
- AdMob

5. Build and Run.
